landing page

al hacer la pagina web, tardo aproximadamente 1 semana o 2. eso depende de la fecha que yo le indique ya que estoy trabajando casi todos los dias como programador. porfavor sugiero paciencia, igualmente los proyectos se los mando antes de la fecha dada hacia el cliente. Resumidamente, para hacer la pagina web, utilizo html para hacer la estructura, despues css para los estilos generales, tambien utilizo algunos frameworks como bootstrap y sass para tener menos codigo y sea mas entendible el proyecto, luego uso javascript para darle la parte mas importante a la pagina, creandole estilos y efecto a los botones, y para concluir utilizo react para darle los retoques finales. los metodos de pago son pocos, pero los mas seguros.
